# Классификация кожных заболеваний

## Описание
Веб-приложение для классификации кожных заболеваний (аллергия, ветрянка, норма) на основе фотографий.

## Модель
- Архитектура: ResNet50
- Точность: 88.75%
- ROC-AUC: 0.9607
- Датасеты: DermNet, HAM10000, Fitzpatrick17k, Monkeypox

## Запуск
### Локально
```bash
pip install -r requirements.txt
streamlit run app.py
